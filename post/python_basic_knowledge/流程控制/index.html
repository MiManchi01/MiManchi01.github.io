Python基础知识总结（五）

一、分支结构（选择结构）

python中三大结构：顺序结构、分支结构、循环结构

在顺序结构中多了一个选择的操作

如果明天老师检查作业的话，我就今晚努力完成，否则我就不写

1.1 单分支结构（if）

格式：

if 条件语句（判断条件）:
    代码语句		# 该语句执行，就是当if后面的判断条件成立（True）就执行，否则不执行

实例：

res = input("老师明天检查作业吗（y/n）：")

if res == 'y':
    print("今晚要写作业了。")

1.2 双分支结构（if…else）

else 要么不存在，要么只存在一个

格式：

if 条件语句（判断条件）:
    代码语句		# 该语句执行，就是当if后面的判断条件成立（True）就执行，否则不执行
else:
    代码语句		#当if判断条件不满足时，执行该代码，否则不执行

实例：

res = input("老师明天检查作业吗（y/n）：")

if res == 'y':
    print("今晚要写作业了。")
else:
    print("今晚开黑")

1.3 多分支结构（if … elif … else）

elif 要么不存在，可以又一个，可以又多个

格式：

if 条件语句（判断条件）:
    代码语句		# 该语句执行，就是当if后面的判断条件成立（True）就执行，否则不执行

elif 条件语句（判断条件）:
    代码语句

else:
    代码语句		#当if判断条件不满足时，执行该代码，否则不执行

实例：

res = input("老师明天检查作业吗（y/n）：")

if res == 'y':
    print("今晚要写作业了。")
elif res == 'n':
    print("今晚开黑")
else:
    print("你***在说啥？？")

练习：

# 练习
# 计算bmi指数
'''
<= 18.4 偏瘦
18.5~23.9 正常
24.0 ~ 27.9 过重
28.0 肥胖
'''
height = float(input("请输入你的身高（单位/米）："))
weight = float(input("请输入你的体重（单位/千克）："))
# 计算bmi指数
bmi = weight / height**2
print("你的bmi指数为：", bmi)

if bmi <= 18.4:
    print("年轻人太瘦啦！！！")
elif 18.5 <= bmi <= 23.9: #  bmi＜=23.9 and bmi＞=18.5
    print("年轻人可以喔~")
elif 24.0 <= bmi <= 27.9:
    print("年轻人该注意啦，有点重了哦~")
else:
    print("没眼看")

1.4 pass 语句

pass是一个空语句，就是占位的意思

当流程控制中，有语句没有想好怎么写时就可以使用pass，防止不必要的报错。

if bmi <= 18.4:
    print("年轻人太瘦啦！！！")
elif 18.5 <= bmi <= 23.9: #  bmi＜=23.9 and bmi＞=18.5
    print("年轻人可以喔~")
elif 24.0 <= bmi <= 27.9:
    pass

二、循环结构

循环：抄写；跑圈

循环：可以避免我们的代码的重复性，提高开发效率

2.1 while循环

格式：

while 循环条件:
    代码语句	#当满足循环条件时，就执行否则不执行
    循环条件的更新

如果循环条件没有更新就很容易进入死循环

num = 1

while num <= 100:
    print("阿宸很帅！" , num)
    num += 1

实例：

num = 1
s = 0
while num <= 100:
    s += num        # s = s + num
    num += 1
print(s)

2.2 for 循环

for循环跟while循环不同，for不需要对循环条件进行手动更新

格式：

for 变量 in 可迭代对象（列表，元组）:
    代码语句

实例：

ls = ['阿宸' , '木槿' , '默契' , '依依' , '杏子']

for i in ls:
    print(i)

range（）函数

取某一区间里面连续的整数

起始值从0开始，不包括终点值

实例：

for i in range(1 , 101):
    print("阿宸很帅！" , i)

三、嵌套

分支结构、循环结构的嵌套

3.1 分支结构的嵌套

今天突然想吃粽子了，附近没有专门买粽子的店，想：走到买包子铺 如果有卖粽子的就看一下是甜的还是咸的，如果是甜的就买两个，如果是咸的就买一个；要是没有卖粽子就买两个豆沙包。

示例：

'''
分析：判断包子铺有没有粽子
1、有
    1.1甜 2
    1.2咸 1
2 没有
    豆沙包2
'''
rice = input("有没有粽子（y/n）")
if rice == 'y':
    taste = input('粽子是甜的还是咸的（y/n）')
    if taste == 'y':
        print("买两个粽子")
    else:
        print('买一个粽子')
else:
    print("买两个豆沙包")

3.2 循环嵌套

A：一周五天：每天的早餐：喝一杯豆浆跟两个豆沙包，午餐：一份猪脚饭，晚餐：木桶饭 一周五天都如此 ，连续了四个星期。

for week in range(1,5):
    print(f"第{week}星期")
    day = 1
    while day < 6:
        print(f'第{day}天,早餐：喝一杯豆浆跟两个豆沙包，午餐：一份猪脚饭，晚餐：木桶饭')
        day += 1

3.4 break、continue

break：终止、停止循环

continue : 逃过本次循环，继续执行下一次循环

同学;今天开始要跑圈，每天四公里。操场跑道一圈只有四百米。今天因为热身没有做好，导致跑到一千二的时候脚崴了。

num = 4000 / 400
i = 0
m = 0
while i < num:
    i += 1
    m += 400
    print(f"跑到第{i}圈，第{m}米")
    if m == 1200:
        print("脚崴了，跑不了了")
        break

今天买了十个小笼包 ， 边走边吃，结果吃到第五个的时候掉地上了，算了，继续吃第六个。

for i in range(1 , 11):
    if i == 5:
        print('掉地上了')
        continue
    print(f'吃到第{i}个小笼包')

四、缩进

Python 中使用缩进对代码的结构进行一个控制。

缩进一般使用四个空格（一个tab键），在同一个代码块里面缩进要一致，在同一篇代码里面要么使用四个空格表示缩进，要么使用一个tab键，不要混用
